== Description ==
 A well crafted starter WordPress theme built on top of Bootstrap 5. We created this theme to fulfill just one goal, "A lightweight theme with less code to remove and ready to use with Bootstrap".


== Frequently Asked Questions ==

= Where are unminified version of SASS files?=
Unminified SASS files are stored in 'sass' folder.

= How can I compile Sass files? =
You will need to install VS Code extension named Live Sass Compiler by Glenn Marks.
